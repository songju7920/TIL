- 정의

  > **계층관계**를 표현할수 있는 데이터를 말한다.

  **트리**형태로 데이터를 저장할수 있으며 컬럼과 스키마가 없다는것이 가장 큰 특징이다.
  Document 내에 Field라는것을 정의하고 Field라는 것을 정의하고 Field 내에 다시 key, Value를 저장한다. 대용량 데이터 저장에 적합하며, 다양한 속성이 있는 데이터를 관리하기 유용하다.

- 종류
  - MongoDB
  - Azure Cosmos DB
  - CouchDB
  - OrientDB
